# 🌿 Highpoint Infusion Lab

A personal batch dosage calculator for cannabis infusions.

## What it does

Enter your batch details and get instant mg dosage breakdowns per tablespoon, teaspoon, and cup.

**Inputs:**
- Strain name
- Flower weight (grams)
- Carrier type (butter, coconut oil, olive oil, etc.)
- Carrier amount (cups)
- THC, CBD, and CBG percentages
- Absorption rate
- Decarboxylated or not

**Outputs:**
- mg per tablespoon
- mg per teaspoon
- mg per cup
- Total active mg per cannabinoid across the full batch

## Usage

Open `index.html` in any browser. No installs, no dependencies, works offline.

**Live site:** `https://yourusername.github.io/highpoint-infusion-calc`
